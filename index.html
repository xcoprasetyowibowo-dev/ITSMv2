Google Apps Script & Firebase Exporter

Arsitektur backend production-ready menggunakan Google Sheet + LockService + REST Firestore

Arsitektur Kunci

  • Firebase REST Client: Terintegrasi via UrlFetchApp tanpa library pihak ketiga, meningkatkan efisiensi startup execution time di bawah 1 detik.
  • Anti Race Condition: LockService (ScriptLock) diaktifkan otomatis di server untuk mencegah duplikasi No. Tiket saat pergantian shift.
  • Batch I/O Processor: Tulis massal otomatis data checklist harian ke baris Spreadsheet menggunakan array memori `getValues()` dan `setValues()`.

Langkah Pemasangan di Hotel

  1. Buat Google Spreadsheet baru.
  2. Klik Ekstensi > Apps Script.
  3. Hapus semua kode default, salin modul kode di samping ke editor.
  4. Atur variabel kredensial di file Config.gs sesuai database Anda.
  5. Deploy sebagai Web App dengan opsi akses "Anyone, even anonymous".
